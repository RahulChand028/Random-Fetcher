# Random-Fetcher
Random-Fetcher helps to fetch random data based on url . Data can be infomat of JSON/XML/TEXT
``` 
fetch.php?type=json&repeat=4&variable=names&names=s2mn3mx6
```

    "names": "dss zju"

    "names": "beda hrk"
  },
